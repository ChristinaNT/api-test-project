## Run Tests

Clone the repository
```
https://github.com/ChristinaNT/api-test-project.git
```

Use the package manager `npm` to install dependencies:
```
npm install
```

Don't forget to audix fixes if needed:
```
npm audit fix
```

Run the tests
```
npm test
```
Jenkins File can be used to run on Jenkins.
